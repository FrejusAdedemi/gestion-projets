# GestionProjets

## Serveur de développement

Pour démarrer le serveur en local :
```bash
ng serve
```

Ouvrez ensuite votre navigateur sur `http://localhost:4200/`. L'application se rechargera automatiquement à chaque modification du code source.

## Génération de composants

Pour générer un nouveau composant :
```bash
ng generate component nom-du-composant
```

## Build

Pour compiler le projet :
```bash
ng build
```

Les fichiers compilés seront placés dans le répertoire `dist/`.

## Tests unitaires

Pour exécuter les tests unitaires :
```bash
ng test
```
